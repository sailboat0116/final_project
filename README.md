## 🕹 遊戲玩法

### 🟡 五子棋（Gomoku）
- 棋盤大小：15x15
- 兩人輪流落子（X 與 O）
- 任一方連成五子即獲勝

### ⚫ 黑白棋（Othello）
- 棋盤大小：8x8
- 初始四顆棋子放在中心
- 落子需翻轉對手棋子
- 雙方無法行動時遊戲結束，以棋子數多者獲勝

## 🧠 設計特色

- ✅ **抽象工廠模式**  
  使用 `GameFactory` 介面統一建立不同遊戲的物件（棋盤、規則、玩家），可自由切換不同遊戲模組。

- ✅ **Swing 圖形化介面**  
  使用滑鼠操作，點擊即下棋，支援自動判斷勝負與換手。

- ✅ **模組化架構**  
  各遊戲邏輯分離，方便日後新增其他棋類（如象棋、圍棋等）。
